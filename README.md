## Run Locally

Clone the project

```bash
  git clone https://github.com/mrkhosravian/mrkhosravian.ir.git
```

Go to the project directory

```bash
  cd mrkhosravian.ir
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run develop
```
